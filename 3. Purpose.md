## Purpose of the guide
This document describes how to use Alert Enterprise Guardian IDM APIs to manage or enforce access related functionality. 

The following functions are added to the scope of the document as they are majorly used in an ideal flow of operation:
- Create Access
- Update Access Details
- Reload Access Grid
- Delete Access

The following functions are kept out of scope functions as they will be used occasionally by auditors or integrators:
- Find Access And Verify Details
- Get Access Details
- Check Filter Functionality
- Find Access
- Find Access by int Status
