## Purpose of the guide
This document describes how to use Alert Enterprise Guardian IDM APIs to manage or enforce access related functionality. As the following capabilities are generally used in an ideal operation flow, they have been added to the scope of the document:
- Create Access
- Update Access Details
- Reload Access Grid
- Delete Access

Out of scope capabilities:
- Find Access And Verify Details
- Get Access Details
- Check Filter Functionality
- Find Access
- Find Access by int Status
